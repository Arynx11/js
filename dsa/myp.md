## Sorting 
``` javascript
let numbers = [5, 3, 8, 1, 2];
numbers.sort((a, b) => a - b);
console.log(numbers);
```

## Hashset
```javascript
let set = new Set();
// 2 Method 
set.add()
set.has()
```
- It mostly used to track duplicate element
- Not occurence of element

## HashMap
```javascript 
let map1 = new Map();
// 3 Method
map1.has()
map1.set()
map1.get()
```
- Store in keys-value pair
- access,add,check by specific
